API

yzt-grid

参数 说明 类型 默认值
data 表格数据 对象格式包含content字段的数组 无
columns 表头字段及表格所需属性 Array 无
id 表格唯一标识，用于保存用户操作编辑列后的记忆 String 无
pageSizeValues 设置表格下拉选项数目 Array [10, 30, 50, 100]
showTitle 显示表格标题栏，配合title属性传值(string|TemplateRef)使用 Boolean false
searchTmpl 配合columns字段传入TemplateRef，显示字段下方的搜索框 Array -
mulitipy 是否显示多选框 Boolean false
fixScroll 固定表格高度，显示竖向滚动条 String | Number 无
selection 支持双向数据绑定，单选返回一个对象多选返回数组 - -
load 支持数据懒加载，返回列表参数 - Output
exportCSV 导出数据，返回表格对象及导出方法doExportCSV(grid,data) - Output

columns可用属性

参数 说明 类型 默认值
header 表头名称 String -
field 表头标识 String -
width 单元格宽度 String -
link 内容可点击 String 标准传true-false识别 -
thumbnail 缩略图 String 标准传true-false识别 -

grid-icon标签

自定义单元格图标

参数 说明 类型 默认值
[ui_grid] 所属表格 模板变量 -
outField 图标所在列 String -
outProp 控制图标是否显示属性 String -
#iconTemplate ng-template模板中必传变量，用于表格识别 - -
let- 属性 传属性值获取当前容器上下文的数据，用于识别图标显藏 String -
