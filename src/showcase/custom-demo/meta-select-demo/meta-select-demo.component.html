yzt-meta 字典

使用nz-select组件封装字典选择组件

如何使用 #

输入字典关键字，显示字典，如果只输入数字会根据电话查找对应的字典，否则根据字典模糊搜索

代码演示

默认不传任何属性。

下拉框支持多选功能，并返回数组的形式(配合vauleType属性使用)

使用[customTemplate]传入template模板，可以实现自定义下拉框样式及定制内容

API

department-select

参数 说明 类型 默认值
ngModel 指定当前选中的条目，支持双向绑定 Array（多选）/String/Object（单选） 无
placeholder 输入框默认提示 String 请选择
metaType(字典类型) service（服务类型），repair（维修类型），fhreturn（返货类型）， riskReason（出险原因）， specifyLogistics（指定物流），abnormalDuty（异常责任方），complement（补件-任务类型）， transportCompany（物流公司）， goodsunit（商品单位），abnormalHandleWay（异常处理方式） String abnormalHandleWay（异常处理方式）
valueType 返回字典id、姓名、对象，'id'|'name'|'object' String 'object'
metaMode 设置 Select 的模式,默认单选 'multiple' -
customTemplate 支持自定义公司名称下拉选项，传入模板 TemplateRef -
width 输入框宽度，支持百分比和像素 String | Number 100%
